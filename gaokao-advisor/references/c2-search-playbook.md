# 检索剧本（逐字照抄版 · 全网开放）

本手册告诉执行者**具体敲什么词、去哪个站、怎么翻存档**。把 `学校名` 替换成实际学校即可。每个维度给出"检索动作序列"，按顺序全部执行完，才算这个维度查完。不许跳步。

> 总原则：先内网快搜定位，再外网+存档深挖（尤其安全维度）。墙内查不到≠不存在，删得越干净的事往往越大。

---

## A. 安全维度检索剧本（最重要，必须全步骤执行）

### A1 内网常规搜（先跑这一轮，定位线索）
依次搜索这些词（每个都搜，别只搜一两个）：
```
学校名 坠楼
学校名 跳楼
学校名 自杀
学校名 学生 死亡
学校名 事故
学校名 失联
学校名 连跳
学校名 通报
学校名 处分 教师
学校名 性骚扰
学校名 霸凌
学校名 退学 后悔
```
平台：网页搜 + 知乎站内搜 + 贴吧"学校名吧"内搜。注意：关键负面常被删、被沉、不在站内搜索结果里——不要以为翻两页就能看到。必须按时间排序捞老帖、用 site: 语法全站搜（见L11）、查精品区沉淀帖、上存档站找被删的（见A4）。搜不到不等于没有。

### A2 时间不限定（铁律）
**安全类检索词绝对不许加"2024""2025"等年份**。极端事件常沉在10年前的页面。抓到一起后，用同样的词把时间往前推，至少覆盖近20年，整理成时间线。

### A3 外网深挖（墙内删了，墙外常有完整记录——必做）
1. **X（Twitter）**：搜 `学校名`（中文）+ `坠楼`/`自杀`/`事件`。校园事件在墙外常有完整时间线、现场图、家属发声。有专门存档国内校园事件的账号。
2. **YouTube**：搜 `学校名 事件`/`学校名 坠楼`。视频删除难度大，常能找到事发记录。
3. **Reddit**：搜学校英文名/拼音 + `China university`，以及中文留学生社区。在校生/校友匿名爆料多。
4. **一亩三分地、北美华人等论坛**：搜学校名，有大量"劝退帖"和匿名内幕。
5. **Google 时间限定**：用 Google（非百度）搜 `学校名 坠楼`，在工具里把时间范围设到事发年份区间，能捞到已删新闻的标题和摘要快照。
6. **海外中文媒体**：重大校园事件，海外中文媒体报道常比墙内通报详细。

### A4 存档/快照检索（专治"删了的"——必做）
1. **Wayback Machine**（web.archive.org）：把"学校名贴吧"的URL、相关新闻URL贴进去，看历史快照。删帖删不掉存档。
2. **archive.today**（archive.ph / archive.is）：同上，另一个独立存档库，墙内可访问性有时更好。
3. **中文维基百科**：搜学校词条，看有没有"争议"/"事件"章节；点"查看历史"，被反复增删的段落值得注意——这是百度百科绝对没有的。
4. **百度快照/必应缓存**：搜索结果旁的"快照"，原页面删了快照可能还在。

### A5 留存度差异判定（写进报告）
对比内外网热度：**某事外网讨论很大、墙内几乎搜不到**——这个落差本身是最重的警报，必须明确写进报告"重点核实事项"，提示家长此事可能被压制，需通过家属发声、外网时间线、事发时段存档帖进一步求证。

### A6 司法记录
- **中国裁判文书网**（wenshu.court.gov.cn）：当事人栏输学校全称，看涉诉案件类型（人身损害、生命权纠纷、劳动争议的分布是信号）。
- 注意：文书网近年部分案件不上网，查不到不等于无诉讼。

### A7 心理支持可及性（必须挖到在校生说法，不许只写"有心理中心"）
- 搜：`学校名 心理咨询 预约`、`学校名 辅导员 半夜`、知乎"在学校名就读体验"里翻心理相关段落。
- 官方数据（专职咨询师数、是否有三甲医院绿色通道）从学生工作部网站挖。
- 两者都要，对照写。

## 执行纪律总结（贴在脑门上）
1. 每个维度的检索动作序列**全部跑完**才算查完，不许只跑前两个词。
2. 安全维度**必须**走完 A1→A7，外网和存档不是可选项。
3. 凡能查到的，写成**带来源的结论**；查不到的，先确认是真没有（外网存档都翻过）再写"需实地核实"。
4. 绝不出现"建议您问招生办X"作为某项的最终答案——那是把活儿踢回家长。

## H. 暗规则检索剧本（官方查无此事，全靠舆情拼图）

暗规则的定义：**文件一套、实操一套**。它不会出现在任何官方页面，只存在于过来人的吐槽、匿名爆料和"懂的都懂"里。挖法是固定句式批量搜：

### H1 分配/派遣类（军校、警校、公费师范、定向医学、委培、订单班——这类院校此节为最高优先级）
```
学校名 分配 关系
学校名 分配 黑幕 / 内幕
学校名 毕业 去向 没关系
学校名 选岗 排名            → 官方说法是综合成绩选岗，对照舆情看实操
公费师范生 分配 乡镇 后悔     → 定向师范的真实去向
定向医学生 违约 / 后悔
军校 分配 新西兰             → "新西兰"=新疆西藏兰州，懂行黑话直接当检索词用
```
- 平台：知乎匿名回答是主战场（搜"在XX就读/毕业是什么体验"翻匿名答案）、贴吧老帖、X上的吐槽。
- 判断要点：分配机制纸面规则（综合排名选岗）vs 舆情反映的实操（名额被打招呼截走、好单位内定）。两者落差就是结论本身。
- **报告里必须回答**：没有任何背景的普通孩子，按概率最可能被分到哪里？这是家长真正的问题。

### H2 保研/评奖/入党黑箱
```
学校名 保研 黑幕 / 不公平
学校名 综测 加分 学生干部      → 综合素质分的弹性空间有多大
学校名 评奖 内定
学校名 入党 名额
学校名 推免 面试 内定
```
- 判断要点：保研细则里"非成绩分"（综测、干部加分、辅导员评分）占比越高，黑箱空间越大。占比是可以从推免细则文件里算出来的硬数据。

### H3 火坑导师与师生权力生态（女孩家庭重点）
```
学校名 导师 避雷
学校名 老师 PUA
学校名 师生恋               → 同时查：学校是否有禁止师生恋/师生不正当关系的明文校规
学校名 学院名 导师 压榨
```
- **导师评价网及其镜像**：原"导师评价网"（mysupervisor）被关停后存在多个镜像站和GitHub备份仓库——在GitHub直接搜"导师评价"能找到开源的评价数据备份。这是本科生选实验室/将来保研选导师前的必查库。
- 知乎搜"学校名+学院名+读研体验"，本科导师制学校（书院制、导师制试点）尤其要查。
- 判断要点：①有没有明文禁止师生恋的校规（有=学校至少在制度上设防）；②出过事是处理老师还是劝退学生（和稀泥方向是最重要的信号）；③避雷帖密度横向比同类学校。

### H4 转专业/校招的实操落差
```
学校名 转专业 名额 内定 / 实际
学校名 校招 内推 截留
学校名 三方 强制签           → 逼签三方刷就业率
```

### H5 教学体系新旧（维度五配套，孩子四年学什么的问题）
```
学校名 专业名 培养方案 pdf     → 看文件落款的修订年份，超过5年未修订=警报
学校名 水课
学校名 课程 过时 / 老古董
学校名 专业名 教材            → 教材版本年份
```
- 判断要点：培养方案修订年份是**硬指标**（文件上有日期）；在校生"学的东西业界早不用了"的吐槽密度是软指标；两者交叉。计算机、电子、金融这类快变行业，课程老旧的代价最大。

## A8. 物理环境与身体安全补充剧本（安全维度扩展，全部必查）

### 偷拍与隐私安全（女孩家庭最高关注）
```
学校名 偷拍
学校名 厕所 摄像头
学校名 浴室 偷拍
学校名 床帘 禁止          → 有学校以"消防/集体主义"为名禁挂床帘，隐私争议
学校名 澡堂 隔间          → 公共浴室是隔间还是通铺，女孩隐私刚需
```
- **判断要点（比事件本身更重要）**：出过偷拍事件后，学校是处理加害者，还是"保护性隔离"受害者（调宿舍、劝退处理、压热搜）？后者模式=出事后你孩子会被怎么对待的预演。受害者是否遭遇"有罪推定"式问话，舆情里都有痕迹。

### 食品安全
```
学校名 食物中毒
学校名 食堂 异物 / 食堂 吃出
学校名 食堂 外包          → 外包食堂出事概率和物价都更高
```
- 江西"鼠头鸭脖"类事件的教训：学校和属地监管可能联手定性，**本地通报要对照外网和当事学生原帖看**。

### 实验室安全（理工科孩子必查）
```
学校名 实验室 事故 / 爆炸 / 烧伤
专业名 实验室 安全          → 化学、化工、材料、生物专业是高危区
```
- 高校实验室事故近年三十余起是公开统计过的，化学化工材料类专业的孩子四年泡实验室，这条不是小概率杞人忧天。查该校该学院有没有前科、安全培训是走形式还是动真格（在校生说法）。

### 宿舍物理安全
```
学校名 宿舍 甲醛           → 新建/翻新宿舍当年入住的甲醛争议（2024烟台等多起先例）
学校名 宿舍 火灾
学校名 宿舍 漏水 / 危楼
```
- 规律：孩子这一届如果恰好入住新落成宿舍楼，甲醛检测报告要求学校出示，舆情先查同校先例。

### 治理压力测试：封校管理史
```
学校名 封校 / 学校名 隔离 管理 / 学校名 疫情 物资
```
- 2020-2022年封校期间各校表现是现成的治理压力测试：物资保障、留学生与本部生差别待遇、一刀切程度、学生诉求响应——极端条件下学校怎么对学生，就是出事时它怎么对你孩子。历史虽过去，治理DNA没变。

## K. 官方自曝文件挖掘（学校被迫公开的"体检底单"，多数家长不知道存在）

### K1 巡视整改通报（最高价值，官方自己骂自己的文件）
- 部属高校接受中央巡视、省属高校接受省委巡视，**巡视反馈和整改通报全文公开**，会直接点名：招生腐败、基建利益输送、师德失范、党委治理弱化、违规吃喝。
- 检索：`学校名 巡视 反馈`、`学校名 巡视整改 通报`。读法：反馈点名的问题=官方盖章认定的病灶；对照几年后的整改通报看是否复发。
- 这是任何宣传材料都盖不住的官方负面清单，**每所公办校必查**。

### K2 高校信息公开年度报告（总账本）
- 教育部规定每校每年10月底前发布，按《高校信息公开事项清单》覆盖10大类50项：招生、财务、招标、人事任免、学风建设、学生申诉处理结果。
- 入口：学校官网"信息公开"专栏，搜`学校名 信息公开 年度报告`。重点翻：学生申诉与处理情况、教师争议处理、收费项目全表。

### K3 学生手册与学籍管理规定（校规原文，孩子要遵守的全部条款）
- 检索：`学校名 学生手册 pdf`、`学校名 学籍管理规定`。这是处分条例、晚归规定、电器违禁清单、床帘政策、休学复学条款的**原文出处**——不要靠舆情猜校规，直接下原文逐条看。
- 重点条款：处分的种类与申诉时限、休学复学的审批条件、留级退学的触发线。

### K4 人民网领导留言板与投诉平台（带官方回复的投诉公开记录）
- 人民网"领导留言板"搜学校名：学生家长投诉（乱收费、宿舍、强制实习）和**属地官方的回复**全部公开，回复态度本身就是治理风格样本。
- 黑猫投诉、12345公开答复同理检索。

### K5 学术诚信国际数据库（不受墙内删帖影响的硬数据）
- **PubPeer**（pubpeer.com）：搜学校名/导师名，论文图像问题的同行质疑记录——曹雪涛事件即由此曝出。报研究强校、孩子未来可能进实验室的，查目标学院。
- **Retraction Watch 撤稿数据库**：查学校撤稿记录，撤稿密度横向比同类校。

### K6 工商司法穿透（民办校与后勤外包必查）
- **天眼查/企查查**搜学校（民办校为企业法人）：股东与实控人、对外投资、欠税记录、被执行人、司法案件——民办校资金链的X光片。
- **中国执行信息公开网**：学校是否失信被执行人。
- 顺藤摸瓜：食堂外包中标公司、宿舍建设承包商的资质与诉讼（招投标信息在信息公开栏和政府采购网）。

### K7 资质验真三件套
- 教育部全国高等学校名单（每年发布）：验野鸡大学，校名一字不差地对。
- 学信网：学籍学历认证体系，录取后可验学籍。
- 教育部涉外监管信息网（jsj.moe.gov.cn）：中外合作项目是否在批准名单。

## L. 平台逐个操作手册（每个平台的正确打开方式）

### L1 B站（视频地面真相，文字会撒谎、视频很难）
- 搜：`学校名 宿舍`、`学校名 食堂 实拍`、`学校名 vlog`、`学校名 开学`。宿舍实拍、食堂菜价镜头、新校区环境一目了然。
- 防坑：招生季的精致"校园vlog"可能是宣传约稿，认准随手拍画质、弹幕和评论区在校生的补充吐槽——**弹幕和评论区往往比视频本身信息量大**。

### L2 GitHub（学生自治知识库，被严重低估）
- 搜：`学校名 生存指南`、`学校名 生存手册`、`大学名拼音 guide`。上交、清华、浙大等多校都有学生开源维护的"生存手册"项目，覆盖选课、给分、保研、宿舍、避坑，**真实度天花板**且不受站内删帖影响。
- 同时搜"导师评价"找评价数据备份仓库。

### L3 贴吧（老帖考古主场）
- 进"学校名吧"用吧内搜索，关键词见各维度剧本；按回复时间排序翻3年以上老帖；"精品区"沉淀帖价值高。
- 防坑：吧务可能被校方收编，置顶帖和删帖痕迹本身是信号；同一ID大量发正面帖=运营号。

### L4 知乎（结构化体验主场）
- 固定题型：`在XX大学就读是怎样的体验`、`XX大学怎么样`、`考上XX大学是什么水平`。**匿名答案单独细读**——敢说真话的都匿名。
- 防坑：高赞不等于真实（招生季有组织点赞），看答案的细节密度（真学生写得出楼栋号和食堂窗口名）。

### L5 小红书（女生视角主场）
- 搜：`学校名 新生`、`学校名 宿舍`、`学校名 避雷`、`学校名 澡堂`。女生关心的隐私、安全、生活细节这里颗粒度最细。
- 防坑：滤镜重灾区，"宿舍改造"帖看改造前原图。

### L6 微信公众号（搜狗微信搜索入口）
- 用搜狗微信搜索搜学校名：校园自媒体、表白墙公众号、家长帮类账号的历史文章，站内搜索引擎搜不到这些。

- "985废物引进计划"等小组站内搜学校名：小镇做题家视角的就读反思、后悔与适应叙事——和知乎的精英叙事形成对照。

## N. 信源可信度分级（拿到信息后怎么定权重）

挖到任何一条，先归档到这个金字塔，决定信几成：

1. **铁证级**：官方公开文件（巡视通报、信息公开年报、裁判文书、学籍政策原文、撤稿记录）。直接采信，但注意官方可能避重就轻。
2. **硬数据级**：可交叉验证的统计（招生计划、推免名单、就业报告雇主名单、天眼查工商）。采信数字，自己做趋势和横向对比。
3. **群体证言级**：多人多年、细节一致的舆情（GitHub生存手册、知乎匿名长答案、贴吧多年同主题吐槽）。**一个人说是个案，五年二十个人说同一件事是模式**——按模式采信。
4. **单点线索级**：单条爆料、Telegram消息、孤立帖子。**只作侦查方向，必须回1-3级验证后才能写进报告**，未验证的标"网传未证实"。
5. **噪音级**：招生季运营号、滤镜种草、有组织点赞的高赞、利益相关的黑稿。**识别并剔除**——招生季的集中正面内容尤其要警惕。

**交叉验证铁律**：负面结论至少要两个独立信源（最好跨内外网）支撑才能下；单一信源的负面只能写成"待核实线索"。正面宣传则反向加压——越是官方主推的，越要找反面证据平衡。

## H6. 事故摆平潜规则检索（最敏感、最该挖的暗规则子项）

挖的是这类口口相传的事：学生出了事故（实验室受伤、校园伤害、坠楼、突发疾病延误等），学校为了**不让闹大、不上新闻、压住舆情**，私下用**保研名额/赔偿金/工作安排/转学**等好处摆平当事人或家属，换取不维权、不曝光、签保密。

**为什么这是最重的治理信号**：它暴露学校处理危机的真实逻辑——不是公开解决和改进，而是**摆平人、捂住嘴**。一所惯于"花钱/给名额消音"的学校，等于告诉你：出了事，它优先保自己的声誉，不是保你孩子。这比"发生过事故"深一层。

### 检索话术
```
学校名 学生 事故 赔偿 保研
学校名 出事 摆平 / 私了 / 封口
学校名 家长 维权 / 闹 / 拉横幅
学校名 压热搜 / 删帖 / 不让上新闻
学校名 实验室 受伤 处理
学校名 学生 死亡 善后 / 补偿
```
- 平台：知乎匿名回答、贴吧老帖、微博（被删的维权)、外网（家属在墙外发声常最完整）、本地论坛。
- 重点找：**家属视角的叙述**（"学校给了XX让我们别闹")、在校生口口相传的"那年那件事最后怎么压下去的"。

### 判断逻辑（看的是处理逻辑，不是事故本身）
- 出事后学校的反应模式：①公开通报+整改+担责（健康）vs ②沉默+私下塞好处+要求保密+压舆情（危险信号）。
- "用保研/工作名额换不维权"这种叙述若反复出现=该校危机处理的惯常套路，记为重点核实。
- 留存度差异在这里尤其关键：家属维权被快速抹掉、外网有内网无，这个落差本身就是警报。

### 呈现纪律（按用户确认的 A 方案：宁可信其有）
- 此类信息几乎全是未证实的口口相传，**一律标注"网传未证实的线索"**，但按风险厌恶原则**不打折地呈现给家长、提示重点核实**——标"未证实"是为诚实，不是为让家长忽略。
- 注明叙述的数量和年份跨度（一条=孤例提示，多人多年同类=模式）。
- 措辞为"检索到以下未证实线索，涉及该校处理事故的方式，建议您通过家属公开发声、事发时段存档、向在校生求证后审慎判断"，给线索和核实方向，不替家长对学校定性。

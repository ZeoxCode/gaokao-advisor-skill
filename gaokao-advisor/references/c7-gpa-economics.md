# 绩点经济学手册——给分/学分/考核如何决定保研与出国

被所有择校产品忽略、但保研党和出国党最该看的一块。核心认知：**同样的努力，在不同学校能换来天差地别的绩点，而绩点直接决定保研资格和出国GPA。** 这不是玄学，是有规律、可查证的。

## 一、给分文化（压分校 vs 松分校）——出国/保研的隐形分水岭

- **压分校的真实代价**：部分名校（复旦、南大、上海交大、浙大等常被点名）以"给分严"著称，均分压得低。**问题在于：出国申请看的是绝对GPA（如3.7/4.0），国外院校不知道也不在乎你的学校给分严，同样85分的能力，在压分校可能只拿到3.3，直接卡在好学校门槛外。** 这就是"复旦给分低、出国吃亏"的真相。
- **保研同理**：保研看校内绩点排名，压分校内部公平（大家都低），但若涉及跨校夏令营、或与松分校学生同台竞争国家奖学金等，绝对绩点低会吃亏。
- **怎么查**：
  - 搜"〔学校名〕 压分""〔学校名〕 给分 好不好""〔学校名〕 GPA 出国"。
  - 出国论坛（一亩三分地、寄托）、知乎"〔学校〕 绩点 出国"有大量过来人现身说法，压分校名单在留学圈是公开的。
  - 看该校有没有"GPA认证/等级转换"政策——有的压分校会出具官方说明帮学生向国外解释给分严，有的则没有，差别很大。
- **判断**：孩子若铁了心出国/保研，"给分友好度"是一个真实权重项。但不是说压分校不能去——名校平台 vs 绩点友好，是个权衡，要摆给家长，不替他定（见元铁律）。

## 二、学分负担（毕业要修多少、累不累）

- **毕业总学分**：各校差异大，有的160学分宽松、有的190+很重。学分重=课多、自由时间少、刷高绩点更难。搜"〔学校〕〔专业〕 培养方案 总学分"。
- **课程分布**：必修占比、选修自由度、有没有大量"水课"占学分（水课多反而可能是刷绩点机会，也可能是浪费时间，看怎么用）。
- **辅修/双学位成本**：想辅修的，查跨学院/跨校辅修政策和额外学分负担。

## 三、考核方式（决定哪种孩子吃香）

- **一考定生死 vs 平时分占比高**：有的课期末一锤定音（适合考试型选手），有的平时作业/出勤/小组占比高（适合稳定型、怕考砸的）。这影响绩点稳定性，和孩子性格匹配。
- **卷面难度与给分曲线**：有没有"强制分布"（规定只能给百分之多少的A，哪怕大家都考得好也要排名压分）——强制分布的专业，绩点竞争更残酷。
- 搜"〔学校〕〔专业〕 考试 难""〔学校〕 正态分布 给分"。

## 四、挂科与补救（风险与退路）

- **挂科率**：杀手课的挂科率（部分培养质量报告有，或在校生口口相传）。
- **补考/重修/清考政策**：挂了能不能补考、重修要不要交费、清考是否已取消（多数已取消，挂科代价变大）、重修会不会覆盖原成绩（影响刷分）。
- **学业预警/退学线**：累计挂多少学分会被退学警告或退学——这是底线安全（见c1维度手册的退学率）。

## 五、学位授予的隐藏门槛（别临毕业被卡）

- **与学位挂钩的硬指标**：四六级是否卡学位、体测/游泳是否达标才发证、本科论文/发表要求、某些学校的"劳动课/军训"学分。
- 搜"〔学校〕 学位 四六级""〔学校〕 游泳 毕业""〔学校〕 体测 学位"。
- 提前知道，免得孩子学分修满却因一项卡住拿不到学位证。

## 六、把这些联动成判断（不是罗列，是回答家长的真问题）

绩点经济学的终点是回答两个问题，报告要正面答：
1. **"孩子想出国，这学校的给分会不会拖后腿？"** → 综合给分松紧+有无GPA解释政策+过来人出国案例，给结论。
2. **"孩子想保研，这学校这专业拿到保研绩点难不难？"** → 综合保研率(分学院)+给分+考核方式+强制分布，给结论。

**呈现纪律**：给分松紧、挂科率这类多来自学生口碑，按c1的信源分级标注（个案/模式）；压分校名单虽在留学圈公开，仍标"据出国论坛与在校生反映"，不写成绝对定论。给判断、给权重、交还选择权。

## 七、检索速查
```
〔学校〕 压分 / 给分 好不好          → 给分文化
〔学校〕 GPA 出国 吃亏              → 出国吃亏与否（出国论坛重点）
〔学校〕〔专业〕 培养方案 总学分      → 学分负担
〔学校〕 强制分布 / 正态 给分         → 考核残酷度
〔学校〕 挂科率 / 重修 清考           → 挂科风险与补救
〔学校〕 学位 四六级 / 游泳 / 体测     → 学位隐藏门槛
〔学校〕 GPA 认证 等级转换 政策        → 有无帮学生向国外解释给分的官方政策
```

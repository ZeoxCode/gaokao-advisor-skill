# 权威数据源清单（带链接/入口）

## 官方硬数据
| 数据源 | 入口 | 提供什么 |
|---|---|---|
| 阳光高考平台 | gaokao.chsi.com.cn | 招生章程、分省分专业计划 |
| 本科教学质量报告 | 学校官网"信息公开"栏，搜"学校名 本科教学质量报告" | 退学率、师生比、农村生源比、学位授予率 |
| 就业质量年度报告 | 同上，下载近3年 | 雇主名单、升学去向、生源地、薪资 |
| 推免公示名单 | 学校教务处/研究生院官网，搜"学校名 推免 公示" | 保研率（分学院算） |
| 转专业实施细则 | 教务处官网 | 转专业门槛、部分公示通过名单 |
| 部门预算 | 信息公开栏，搜"学校名 部门预算" | 经费体量 |
| 裁判文书网 | wenshu.court.gov.cn | 涉诉案件 |
| 选调生名单 | 搜"XX省 定向选调 高校名单" | 政府认可度 |
| 学科评估/双一流 | 教育部学位中心、双一流名单 | 学科水位 |

## 社交平台（舆情）
| 平台 | 入口 | 看什么 |
|---|---|---|
| 知乎 | "在XX大学就读体验"、"XX大学 劝退" | 校风、学风、专业地位（核心） |
| 百度贴吧 | "学校名吧"内搜 | 日常吐槽、老帖（删得慢） |
| 小红书 | "XX大学 宿舍/避雷/新生" | 女生视角生活细节 |
| 微博 | 搜学校名+按时间排序 | 事件时间线 |
| BOSS直聘/猎聘 | 搜"学校名 专业" | 真实薪资 |

## 外网与存档（安全维度必用）
| 工具 | 入口 | 用途 |
|---|---|---|
| X (Twitter) | 搜学校中文名+事件 | 校园事件完整时间线、家属发声 |
| YouTube | 搜学校名+事件 | 难删除的事件视频 |
| Reddit / 一亩三分地 | 搜学校名 | 校友匿名爆料、劝退帖 |
| Wayback Machine | web.archive.org | 网页历史快照（删帖删不掉存档） |
| archive.today | archive.ph / archive.is | 独立存档库 |
| 中文维基百科 | zh.wikipedia.org | 学校词条"争议"章节 + 编辑历史 |
| Google 时间限定 | google.com 工具→时间范围 | 捞已删新闻的快照摘要 |

## 部委血统速查（1998年前）
电力部→华北电力/上海电力/东北电力/长沙理工；铁道部→西南交大/北京交大/兰州交大/石家庄铁道；邮电部→北邮/南邮/重邮/西邮；财政部央行→上海财大/中央财大/西南财大/河北金融学院；税务→各地税务学院；海关→上海海关学院；民航→中国民航大学/中飞院。验证靠就业报告雇主名单。

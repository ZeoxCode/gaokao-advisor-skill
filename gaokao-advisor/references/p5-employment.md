# 就业手册——家长的终极问题"孩子出来有没有路"

报志愿一切计算的终点都是就业。本手册帮家长把"四年后孩子靠什么吃饭"想清楚。注意：行业景气是动态的，手册给的是判断框架和当下快照，执行时必须现场查最新数据，不照搬。

## 〇、就业地域：绝不等于"学校所在城市"（纠正机构视角的核心）

招生宣传默认"在哪读≈在哪就业≈在哪定居"——这是学校所在地的口径，不是真实就业流向。真实情况，就业地域由**三种流向**决定，必须按专业类型判断孩子大概率走哪条，不许一概而论：

1. **跟着对口行业的全国布点走（行业校最典型）**：
   - 铁道院校→全国各地铁路局（哪有铁路局去哪，不是留在学校所在城市）；电力校→各省电网公司；矿业/石油→全国矿区油田；民航→各大航司基地；师范部属校→全国招教。
   - 判断：原部委血统校、行业特色校，就业看的是**对口系统在全国的布点**，学校在哪只是读书的地方。一所中部的铁道院校，毕业可能去广铁、上海局、乌鲁木齐局——遍布全国。
2. **回生源地（师范、医学、考公、本地国企倾向的孩子）**：大量孩子读完回老家就业考公，尤其师范进家乡教师编、医学回本地医院、考公回生源地。这种情况学校所在城市只是过客。
3. **留校所在城市/城市群（市场化就业、互联网/金融/外企倾向）**：这种才符合"在哪读留哪"，主要是去大城市进民企外企大厂的孩子。

**给家长落地**：先判断孩子走哪条流向（行业校→看对口系统布点；想考公回家→看生源地；想market化→看学校所在城市群），再谈就业地域。**绝不简单说"这学校就业在XX市"。**

## 一、看"孩子毕业那年"的行业，不是今天的行业

- 报考看的是4年后的就业市场。今天的热门可能4年后过剩（前车之鉴：土木曾是香饽饽、生化环材曾被吹捧），今天的冷门可能逆袭。
- 判断行业景气的三个信号：①国家战略方向（十四五/十五五重点扶持的=有政策红利，如AI、半导体、新能源、高端制造、生物医药）；②人才缺口数据（招聘平台的供需比）；③头部企业是扩招还是裁员。
- 当下快照（需现场更新）：AI/算法岗持续高热、缺口大、起薪过2万；央国企也在抢AI人才（薪资略低但稳定、解决户口）；传统土木/低端制造/教培收缩；金融两极分化（头部光鲜、普通院校一般）。

## 二、四条主要就业路径（帮孩子的性格和家庭对号入座）

1. **市场化就业（大厂/民企/外企）**：薪资上限高、凭本事、机会多；代价是卷、不稳定、有35岁焦虑。适合能力强、想搏、抗压的孩子。对应专业：计算机、电子、金融、设计等。
2. **体制内（考公/考编/选调）**：稳定、有编制、社会评价稳；代价是收入封顶、按部就班、有的地方关系文化重。适合求稳、家庭看重编制的孩子。**专业是硬门槛**——见下方考公对口表。
3. **国企央企校招**：介于前两者之间，比公务员市场化、比大厂稳定，常解决户口。电网、烟草、三大运营商、各大行、军工集团等，**对口院校（行业血统校）和专业有明显偏好**（见学校体检的血统通道）。
4. **深造后再就业（考研/读博/出国）**：很多专业本科不值钱，必须读研才有竞争力（生化环材、心理、基础理学、部分文科）。把"至少读到硕士"的时间和金钱成本提前算进决策。

## 三、考公考编的专业对口真相（求稳家庭必看）

- 公务员招录大量岗位限制专业。**友好专业**（能报的岗位多）：法学、财政学/会计/审计、经济学、计算机、汉语言文学、行政管理、统计学、税务。**不友好专业**（三不限岗位扎堆、竞争惨烈）：很多工科、冷门理学、艺术、小语种。
- 想走体制内，选专业时就查目标省考/国考的"专业分类目录"，看这个专业能报多少岗位。"三不限"岗位（不限专业）往往几千人抢一个，是地狱模式。
- 选调生（尤其定向选调）对学校层次和专业双重要求，是体制内最优路径，见学校体检的选调名单部分。

## 四、各大类的就业面貌速写（性格速写，具体专业仍要逐个查）

- **计算机/电子/AI**：出口最宽、起薪最高、可去各行业；卷、吃天赋、35岁焦虑、有行业周期。
- **金融/经管**：头部光鲜（投行/券商/基金），但极度看出身院校和资源，普通院校就业普通；考公考编友好。
- **医学**：稳定受尊重、越老越吃香；战线超长（5+3+规培）、累、医患压力，要真热爱能熬。
- **法学**：红牌但有门槛（法考），过了稳、过不了尴尬；考公友好、看出身。
- **师范**：稳定有编制吸引力，但教师岗渐饱和、师范生过剩苗头；看学科（数理化生缺口大于文科）和地区。
- **理学（数理化生）**：本科不直接对口，基本要读研走科研，或转码/转金融。
- **工科（机械/电气/化工/材料/土木等）**：差异极大——电气（电网）、自动化稳；土木退潮、化工材料属"生化环材"天坑需深造。必须逐个专业查，不能一概而论。
- **农林**：被低估，部分有定向/选调/编制优势，不嫌弃反而捡漏。
- **文史哲**：就业面窄，多需考公考研深造，适合目标明确的孩子。
- **艺术设计**：撤销重灾区、对口率低、同质化严重，非真有热情和资源慎入。

## 五、AI冲击视角（这一代孩子四年后要面对的新变量）

- 重复性、标准化、纯信息处理类岗位（基础翻译、基础文案、初级数据处理、部分会计核算）受AI冲击大。
- 抗冲击的方向：需要人际信任与责任承担的（医生、律师、教师）、需要动手和现场的（很多工科、手术、工程）、AI本身的生产者（计算机/AI/数据）、需要复杂创造和审美判断的高端创意。
- 提示家长：别只看专业"现在热不热"，也想想它"会不会被AI替代大半"，选偏向"和AI协作"或"AI难替代"的方向更稳。

## 六、就业信息源
- 麦可思《中国本科生就业报告》（红黄绿牌、对口率、薪资、满意度）。
- 各校《毕业生就业质量报告》（具体雇主名单、分专业去向）。
- 智联/BOSS/猎聘的行业薪资和供需报告。
- 国家公务员局、各省人事考试网的专业分类目录（考公对口）。
- 各央国企校招官网历年录用公示（看行业通道）。
- 脉脉、知乎"XX专业就业""XX行业现状"看从业者真实声音。

## 七、考公考编专业对口表（求稳家庭按此反推专业）

当孩子/家庭以考公考编为核心目标时，选专业要"反着来"——**先看哪些专业未来能报的岗位最多，再倒推现在选什么**。这张表要完整呈现，不能只给三五个常规选项。

**招录大户专业（能报岗位最多，强烈推荐求稳家庭考虑）**：
- **法学**：法院、检察院、司法、纪检监察的刚需，岗位极多；但需通过法考。
- **会计/审计/财政学**：各级机关财务岗、税务、审计署系统的硬通货，招录量大且稳定。
- **汉语言文学/中国语言文学**：党政机关综合文秘岗的第一大户，"笔杆子"刚需，岗位海量。
- **经济学/金融学**：发改、财政、央行、银保监等经济管理部门。
- **计算机/网络安全**：各机关信息化岗、网信、公安网安，需求逐年涨。
- **统计学**：统计局及各部门数据岗。
- **政治学/行政管理/思想政治教育**：党政机关、组织部、宣传系统对口（**选了政治的考生在这块有先天优势**）。

**考公不友好专业（能报岗位少，多挤"三不限"地狱岗，千人一岗）**：
- 多数纯工科、冷门理学（数理化生本科）、艺术、哲学、历史学、小语种、农林（部分）。
- "三不限"岗位（不限专业/不限户籍/不限基层经历）竞争比常达上千比一，是地狱模式。

**选科与考公的隐藏关联（务必点透）**：
- 选了**政治**的理科生（如物化政组合），未来既能报理工岗，又能报政治学/行政管理/思政这类党政对口专业，是"理科生里最适合考公"的身位，这层优势要完整告诉家长，不止用在警校门槛上。

**操作**：想考公的孩子，选专业前先查目标省/国考的"专业分类目录"，看候选专业能报多少岗位，按"岗位数"而非"专业听着好不好"来排序。

## 八、医学类专业的"资格与编制"真相（医学方向专属必查）

家长奔着"医院系统稳定"去时，必须挖到这一层，不许停在"在医院工作"：
- **医师 vs 技师的天壤之别**：临床医学（5年制）毕业考执业医师、能看病开方写诊断；而**医学技术类（影像技术/检验技术/康复治疗等，4年制）毕业是"技师"，能操作设备做检查，但不能写诊断报告、不是医生**。职业天花板、薪资、社会认可差一大截。
- **编制现实**：三甲医院技术岗编制收紧，本科医技类进好医院常是合同制或被规培卡，"就业稳"要打折扣。
- **必须点破的落差**：选物化生、想进医院、但位次只够医技类够不着临床的孩子——要明确告诉他"你够得着的是当技师不是当医生，这两者差别很大，你要的是哪个？"（见报志愿主SKILL元铁律：模糊诉求要点破落差）。
- **护理**：就业稳但劳动强度大、夜班、医患压力，女孩家庭要如实告知现实。
- 检索："医学技术 技师 不能写报告""〔专业〕 就业 天花板 规培"。

## 九、带名字的雇主情报（学生口口相传的硬通货，就业模块的精华）

宏观趋势（哪个行业涨跌）只是底图。家长和学生真正反复看的，是**带公司名字的口碑情报**——哪家钱多、哪家坑、哪家毁约、哪家是外人不知道的宝藏。这层不挖，就业就是浅的。

### 核心挖法：去公开平台捞"外流/搬运"的内容
封闭校内BBS（北邮人byr、水木清华、日月光华复旦、小百合南大等）是这些情报的源头，但**有价值的内容一旦发出就会外流到公开平台**——有人截图发微博、搬运到知乎、整理成公众号文章、贴到牛客/一亩三分地、存到网盘。所以：
- **优先在公开平台搜外流版本**，把封闭论坛名当关键词用：
```
北邮人 雇主红黑榜
水木 offer 红黑榜 / 水木 薪资
学校名 BBS 就业 红黑榜
学校名 雇主 红黑榜 / 避雷
学校名 offer 比较 / 学校名 秋招 总结
```
- 封闭BBS只作"知道有这源头"的线索；原帖搜不到再托在校生，但**多数有价值内容公开平台就能捞到**，不必卡在账号墙前。

### 四类要挖的雇主情报
1. **雇主红黑榜**：学长学姐发的"哪家公司钱多事少/坑/面试体验差/管理烂"的口碑沉淀。最有本校针对性。
2. **薪资爆料表**：应届生自发整理的offer薪资共享表。源：**OfferShow、牛客网、一亩三分地、应届生求职网**，搜"〔学校/专业〕 offer 薪资 2025"、"〔行业〕 应届 薪资 表"。
3. **小众宝藏雇主（最易被忽略、最值钱）**：不是华为腾讯这种谁都知道的，而是**"某研究所/某地方国企/某细分行业隐形冠军，专门偏爱某几所学校、待遇好又稳定、但外人根本不知道"**——如各类设计院、电网省/市公司、烟草、三大运营商研究院、军工院所(航天X院某所)、各部委直属事业单位。这类和"血统通道"强绑定（见c5），是某些学校"性价比远超其排名"的真相。挖法：扒该校就业报告里反复出现的非知名单位、问校友"咱们学校有哪些外人不知道但很香的对口去处"。
4. **雇主黑名单**：爱毁约的、"大小周血汗厂"、PUA严重的、裁员狠的公司——学生口口相传的避雷名单。搜"〔公司〕 毁约 应届""〔公司〕 避雷 牛客"。

### 判断与呈现纪律
- 红黑榜是主观口碑，按信源分级标注：一两条吐槽=个案，多年多人共识=可信。给情报，不替孩子定"必去/必避"。
- **薪资必须给分布，禁止天花板冒充预期**：报道里"某专业起薪X万"常是头部极值。必须区分"头部少数（金字塔尖、需顶尖+读研）/中位数/大多数人去向"三档，给家长真实的薪资分布，不给一个能麻痹人的高数字。
- 薪资数据注明是"网传爆料、非官方"，给区间不给绝对值。
- 小众宝藏雇主要交叉验证就业报告确认"确实招这校"，不能只凭一句传言。

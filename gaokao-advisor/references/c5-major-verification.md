# 专业核实手册（核实层面：这所学校的这个专业到底什么成色）

分工：报志愿 skill 管"选哪个专业方向"；本手册管"选定的专业，在这所具体学校，是不是亲儿子、值不值得来"。同一个专业名，在不同学校天差地别，本手册就是查这个差别。

## 一、这个专业在这所学校的"地位"——亲儿子还是后娘养的

同名专业在A校是王牌、在B校是凑数的，决定因素：
- **是不是建校老专业/特色专业**：学校的看家专业，资源、师资、校友网都倾斜；新设的"蹭热点"专业（很多学校跟风开的人工智能、大数据）可能师资东拼西凑。
- **有没有硕士点、博士点**：有博士点=学科完整、保研留本校有路、师资强；只有本科无硕博=师资和深造受限。查：研招网搜该校该专业硕博点。
- **学科评估等级**：该专业所属一级学科在教育部评估的档（A/B/C），是该校这个专业实力的官方标尺。
- **国家级一流专业/特色专业/卓越计划**：教育部和省的专业级头衔（区别于学校级头衔），是专业含金量信号。
- **大牌教授给不给本科生上课**：名册上的院士长江学者，是真上课还是只挂名。挖在校生说法。

## 二、教学体系新旧（孩子四年实际学什么）

- **培养方案修订年份**（硬指标）：搜该校该专业培养方案PDF，看落款修订年份。超5年未修订=警报，计算机/电子/金融等快变行业尤甚。
- **教材版本**：用的是最新教材还是十年前的老古董。
- **课程与业界脱节度**：在校生"学的东西业界早不用了""水课多"的吐槽密度。
- **实践与实验条件**：实验设备新旧、实习基地质量、有没有真项目可做——工科尤其看这个。
- **校企合作/产业学院**：和业界头部企业有没有联合培养、定向班（可能是就业直通车，也可能是廉价劳力陷阱，结合体检的强制实习项）。

## 三、这个专业在这所学校的出路（最硬的核实）

- **分专业就业去向**：《就业质量报告》里能拆到专业的，看该专业具体去了哪、对口率多少。
- **分学院保研率**：保研率必须看到学院/专业层级，不是校均值——同校不同专业能差一个量级（见学校体检01手册）。
- **考研去向**：该专业学生考研都去了哪些学校，本校保护力度。
- **该专业的真实薪资**：招聘网站搜"学校名+专业名"看毕业生岗位报价，不信就业报告平均数。
- **校友混得怎么样**：脉脉、领英搜该校该专业校友的职业轨迹。

## 四、这个专业本身的风险（叠加报志愿阶段的红牌判断，落到具体校）

- **本校该专业是否在撤销/缩招预警名单**：教育部和该校拟撤销公示里有没有它；近年招生数是否逐年缩水（缩招=可能要撤=师资在撤退、学弟妹断档、保研点将消失）。
- **大类招生分流的真实处境**：如果是大类招进去再分流，该校分流规则（成绩还是志愿）、目标专业分流难度、最差被分到什么——高分进大类可能被挤到冷门方向。
- **转专业进出该专业的难度**：想转进这个热门专业有多难（接收名额、考核）、从这个专业转出容易吗（万一不喜欢的退路）。出口数据见学校体检01手册。

## 五、资质认证（决定文凭含金量的隐藏开关）

- **工程教育认证**（华盛顿协议）：工科专业过没过，影响工程师执业资格和国际互认。名单公开。
- **专业认证**：医学（临床医学专业认证）、师范（师范专业认证）、法学等，过没过、是否带条件通过，影响规培/资格考/就业。
- **中外合作专业的留服认证**：能否做教育部留服认证，几十万学费的天壤之别，涉外监管网查。

## 六、专业核实检索剧本（逐字照抄）
```
学校名 专业名 培养方案 pdf            → 修订年份+课程
研招网 学校名 专业名 硕士点 博士点      → 学科完整度
学校名 专业名 学科评估 / 一流专业       → 官方头衔
学校名 专业名 就业 / 保研              → 出路（分专业）
学校名 专业名 水课 / 过时 / 劝退        → 在校生吐槽
BOSS直聘 学校名 专业名                 → 真实薪资
学校名 专业名 撤销 / 缩招 / 招生人数 历年 → 存续风险
工程教育认证 / 专业认证 名单 + 学校名专业名 → 资质
学校名 专业名 大类 分流 规则           → 分流处境
```
- 核心信源：研招网（硕博点）、该校教务处（培养方案）、就业质量报告（分专业去向）、知乎"XX大学XX专业就读体验"、招聘网站（薪资）。

## 七、查专业独立成模块：两层分开又衔接（整改8）

查专业是和查学校并列的第三根支柱，不是附属。它有两层，必须分清又衔接：

- **第一层·专业本身好不好**（选择层，方法见报志愿侧 p2-select-major）：这个专业学什么、就业宽窄、本硕博必要、红牌与否、行业趋势。回答"要不要学这个专业"。
- **第二层·这校的这专业成色**（核实层，本手册）：同一个专业名，在这所具体学校是什么水位、什么处境。回答"既然要学，在这家学值不值"。
- **衔接**：先用第一层判断"专业方向对不对孩子"，再用第二层判断"这所学校的这个专业配不配得上"。两层都过关才推荐。报告里两层都要呈现，不混为一谈。

## 八、专业内部的方向分化：哪个方向是金、哪个是坑（整改9）

同一个专业，内部方向天差地别，必须挖到方向级：
- **计算机**：搞AI/算法（吃学历、天花板高）vs 搞嵌入式/硬件 vs 搞理论 vs 搞运维测试——出路和钱景差很多。
- **临床医学**：未来进哪个科室（外科/内科/影像/麻醉/儿科/急诊）前景、强度、收入差异巨大。
- **机械/材料**：偏传统制造 vs 偏半导体/新能源方向，一个退潮一个风口。
- **金融**：量化/投行/银行柜员，同一专业两极分化。
- **挖法**：搜"〔专业〕 方向 区别 就业""〔专业〕 哪个方向 好"；看该校该专业的**分流方向设置**（大二大三往哪些方向分）；问在校生"咱们专业最好和最坑的方向分别是哪个"。
- **报告要落地**："这个专业在这校可分X/Y/Z方向，其中〔方向〕出路最好但竞争激烈、〔方向〕相对边缘，分流在大〔几〕按〔成绩/志愿〕进行。"

## 九、课程级颗粒度：查专业要比查学校更细（整改10）

- **培养方案逐项看**：核心课是哪几门、决定专业上限的"硬课"是什么、有多少"水课"占学分、有没有"挂科重灾区"。搜"〔学校〕〔专业〕 培养方案 pdf"逐条读。
- **课程新旧**：修订年份、教材版本、课程与业界脱节度（快变行业如计算机/AI，老旧课程代价大）。
- **实践条件**：实验设备、实习基地、有没有真项目——工科尤其看。
- **报告落地**："该专业核心课为〔列举〕，其中〔某课〕是公认硬课/挂科点；培养方案〔年份〕修订，〔评价新旧〕。"——细到孩子知道未来要啃哪几块硬骨头。

## 十、校级专业级就业出口：落到"这校这专业的人都去哪了"（整改11）

- 不停在行业级泛谈，要挖到这所学校这个专业的**具体出口**：去年毕业生具体去了哪些单位/读研去了哪/对口率多少/起薪多少。
- 源：该校《就业质量报告》拆到专业的部分、该专业公众号/学院网站的就业喜报、脉脉领英搜该校该专业校友轨迹、招聘网站搜"〔学校〕〔专业〕"。
- 报告落地："据该校就业报告/校友信息，该专业毕业生主要去向〔具体单位/院校〕，对口率约〔X〕，起薪约〔Y〕。"
- **往下挖一层·小众宝藏雇主**：不止看华为/电网这种知名去向，重点扒该专业就业名单里**反复出现的非知名单位**——某研究所、某地方国企、某细分隐形冠军，可能正是这所学校"外人不知道但圈内认、待遇好又稳"的对口宝藏（与c5血统通道、p5第九节联动）。这类雇主往往是某校"性价比远超排名"的真相，必须挖出来告诉家长。

## 十一、专业的"劝退真相"：过来人最后悔/最想提醒什么（整改12）

- 专挖匿名爆料里的"过来人忠告"：搜"〔专业〕 劝退 真相""〔学校〕〔专业〕 后悔""〔专业〕 千万别报/还好报了"。
- 不是简单贴"天坑"标签，是呈现"过来人具体后悔什么、具体提醒报考者注意什么"——可能是"以为是计算机其实是数学"、"以为进医院其实是技师"、"课程太水学不到东西"、"考研不歧视但就业第一学历卡得死"。
- 按信源分级标注（个案/多人多年的模式），给真相不替孩子裁决。

## 十二、专业对比功能（整改13）

支持两类对比，对比时按家庭权重高亮差异，不算总分：
1. **跨校同专业**："A校计算机 vs B校计算机"——比学科评估、保研率、给分、就业出口、城市、方向设置。
2. **同校相近专业**："这校的 计算机 vs 软工 vs AI"——比课程、分流、出路、难度，帮怕选错方向的孩子看清差别。
- 对比表列出关键差异点并标注"对你家而言，X维度上〔哪个〕更优（因为你看重〔家长输入〕）"，给倾向、交还选择。

## 十三、下行期评专业：用上岸指标替换盛世指标（关联p7）
评这校这专业，别只用就业率/薪资（盛世注水指标），下行期换成：①上岸率（考编/考研/进体制的真实比例，非"灵活就业"）②裁员安全度（35岁会不会被优化、越老越值钱否）③学历贬值速度（本科值钱吗、是否必须卷到博）④第一学历抗跌性（这校牌子在下行期求职/考公的硬度）。详见 p7-downturn-survival。

## 十四、就业出口硬红线（治"泛"，缺一不合格，自检必查）

专业核实里最容易偷懒泛化的就是就业。下面是不合格 vs 合格的对照，一律向右看齐：

- ❌"去大厂当数据分析师、去保险公司做精算、量化私募抢着要起薪30-50万"（类别+悬空数字+天花板冒充预期）
- ✅"据该校该学院〔年份〕就业报告/校友信息：本科直接就业约X人，主要去〔具体公司名各几人〕，起薪中位数约〔X〕；读研后出口另算。量化/精算等高薪去向是金字塔尖极少数（约前X名），全专业薪资中位数其实是〔Y〕，大多数人去向是〔Z〕。"

**四个硬要求，缺一不合格**：
1. **具体到公司名+大致人数**：不是"大厂""券商"，是"腾讯X人、国寿Y人、幻方Z人"。挖该校该专业就业报告的雇主名单、就业喜报、校友轨迹。
2. **薪资给分布不给天花板**：禁止用头部极值（量化50万）冒充专业预期。必须给中位数+大多数人的真实去向。头部高薪要标"金字塔尖少数人、需读研+顶尖才够得着"。
3. **本科出口和读研出口分开**：升学率高（如65%）意味着只有少数本科直接就业——必须单独说清"不读研的那拨人真实去哪、什么水平"，不许用升学高光盖住本科出口。
4. **励志断言必须验证**："纯凭本事不看关系""考证就被抢走"这类顺耳的正面判断，按"正面结论必须有据"挖在校生/从业者实证；挖不到就标"网传，待核"，不当结论给。功利视角下，越励志的判断越要警惕，它最能麻痹家长。

**自检反问**：就业出口若家长看完还是不知道"我孩子大概率去哪、拿多少、不读研行不行"，就是泛了，退回重挖。

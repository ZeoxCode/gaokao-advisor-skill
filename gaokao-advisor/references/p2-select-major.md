# 选专业手册（选择层面：选哪个，不是查某校的某专业）

注意分工：本手册管"该往哪个专业方向走"。"某校的某专业成色如何"归学校体检 skill 的专业核实模块。

## 一、破除家长最常见的三个认知误区（先纠偏再选）

1. **"热门=好就业"是滞后信息**。家长记忆里的热门是10年前的热门。计算机、金融曾是王道，但风口会变、会过剩。报考要看的是"孩子毕业那年(4年后)的行业景气",不是今天的。绿牌(走高)和红牌(走低)是动态的。
2. **"冷门=没前途"也未必**。被劝退多年的环境、生态、文物修复、殡葬、地质等,部分因国家战略转向或供给稀缺而逆袭。冷门里有真坑也有被错杀的金矿,要分辨。
3. **"名字好听=专业好"是最大的坑**。信息管理与信息系统(撤销最多)、生物医学工程(听着高大上实则就业窄)、公共事业管理(连续红牌)——很多名字唬人的专业是天坑。**永远看课程设置和毕业去向,不看专业名。**

## 二、红牌/天坑专业预警(动态查证,不照搬名单)

- **权威源**:麦可思《中国本科生就业报告》(就业蓝皮书)每年的红黄绿牌榜 + 各省教育厅发布的就业率低专业预警清单(2023年起教育部要求各省报送)。
- **近年高频红牌**(供参考,每年更新):公共事业管理、音乐表演、绘画、法学、美术学、信息管理与信息系统、部分艺术设计类。**法学连续多省警示、绘画连续五年红牌**。
- **判断逻辑而非背名单**:一个专业是不是坑,看三件事——①近5年被红牌/预警的次数(≥4次警惕);②全国近10年的撤销布点数(撤得多=在退潮);③对口就业率(部分艺术类对口率不足30%)。三条都中=真天坑。
- **撤销名单是硬数据**:教育部和各校每年公示拟撤销专业,孩子要避开"读着读着专业就没了"的(师资撤、保研点消失、学弟妹断档)。

## 三、专业的"全貌画像"——选之前必须搞清的6件事

不要只看一个专业名,每个候选专业都拉出这6项:
1. **到底学什么**:核心课程是什么(数学物理要求多高?编程多不多?背诵量大不大?)。很多人填了才发现学不下去——生物要背、计算机要数学、医学要五年+规培。
2. **毕业后干什么(典型一天)**:主流对口岗位每天实际做什么、在什么环境(写字楼/工地/实验室/医院/学校)。复用学校体检 skill 的"毕业后典型一天"方法,这里是行业层面的。
3. **就业面宽窄**:出口是多元(如计算机可去各行业)还是单一(如某些工科只对口一两个行业,行业一冷全完)。
4. **天花板与起薪**:行业薪资中位数、晋升路径、35岁现象严重不严重。看招聘网站真实JD不看宣传。
5. **考研刚需度**:有些专业本科不值钱、必须读研(生化环材、心理、部分文科),要把"至少读到硕士"的时间金钱成本算进决策。
6. **考公考编友好度**:专业是否进主流招录目录(法学、财会、计算机、汉语言友好;很多工科和冷门专业能报的岗位极少)。这对求稳家庭是硬指标。

## 四、孩子这个人(选专业最该问、最少人问)

- **兴趣与忍耐**:不是"喜不喜欢",是"学不下去的那部分能不能忍"(喜欢医但怕不怕血和尸体?喜欢码农但能不能忍长期高压debug?)。
- **性格匹配**:内向能不能做要大量沟通的(销售/律师/教师)?坐不住的适不适合科研?
- **能力底子**:数学差硬冲金融工程/计算机会很痛苦;手眼协调差别碰外科/口腔。诚实评估,不自欺。
- **家庭资源**:某些专业(医学、法律、艺术、建筑)出成绩慢、前期靠家庭托底,或回报高度依赖人脉资源,要掂量家底。
- **底线**:分数是孩子的,人生是孩子的。家长可以否决明显的坑、可以提供信息和阅历,但**不该违背孩子强烈意愿包办**——清华化工状元后悔10年是反面教材。skill 要同时服务家长的风险控制和孩子的意愿,冲突时摆明利弊、把选择权交还。

## 五、专业大类的常识地图(给茫然家长的入门底图)

按"家长+孩子"实用视角速记各大类的真实面貌(非排名,是性格):
- **计算机/电子/软件**:就业宽、起薪高、但卷且吃天赋、35岁焦虑、行业有周期。
- **金融/经济**:光鲜但头部看学历出身和资源,普通院校就业一般,去向两极分化。
- **医学**:稳、受尊重、但战线超长(5+3+规培)、累、医患压力,要真热爱+能熬。
- **法学**:红牌但有门槛(法考),通过后稳,通不过尴尬,出身院校很重要。
- **师范**:稳定、有编制吸引力,但教师岗趋饱和、师范生过剩苗头,看学科和地区。
- **理学(数理化生)**:基础学科,本科不直接对口,基本要读研走科研或转码/转金融。
- **工科(机械/土木/化工/材料等)**:对口实体行业,有的稳(电力)有的退潮(土木),个体专业差异极大,必须逐个查不能一概而论。
- **农林**:被低估,部分有定向/编制/选调优势,真不嫌弃反而捡漏。
- **文史哲**:就业面窄、多需考公考研深造,适合目标明确(学术/考公/文教)的孩子。
- **艺术/设计**:撤销重灾区、对口率低、同质化严重,非真有热情和资源慎入。
**(以上是性格速写,任何具体专业仍要按第三节6件事逐项查证,别用速写代替调查。)**

## 六、选专业信息源
- 麦可思就业蓝皮书(红黄绿牌、薪资、对口率)、各省就业率预警清单。
- 阳光高考"专业库"(开设院校、课程、就业方向官方说明)。
- 招聘网站(BOSS/猎聘)搜专业名看真实岗位与薪资。
- 知乎"XX专业 劝退/就业""XX专业 后悔"、B站专业解读、各专业从业者自述。
- 教育部专业撤销/新增名单(动态趋势)。

## 七、专业深度档案（选定方向后，往下钻一层）

选专业最大的坑是"只看专业名、不看里面"。每个候选专业要钻到这一层：

### 同名专业天差地别——必看细分方向
- **计算机类**：计算机科学与技术、软件工程、人工智能、网络安全、数据科学、物联网——名字都沾"计算机"，但课程、就业、薪资差异大。软工偏工程落地、AI偏算法研究（更吃学历）、网安偏安全攻防（体制内需求大）。
- **电子信息类**：电子科学与技术、通信工程、微电子（芯片）、集成电路——微电子/集成电路是国家急需、政策重仓，和通信工程出路不同。
- **医学类**：临床、口腔、麻醉、影像、检验、护理、基础医学、预防医学——临床口腔是"正统"、护理预防出路差很多，别被"医学"两字笼统吸引。
- **经管类**：金融、会计、经济学、财政、国际贸易、工商管理——会计务实好就业、工商管理偏空泛、金融看出身。
- **教训**：永远查这个专业的**具体培养方案和核心课程**，而不是凭专业名想象。

### 课程难度与个人匹配（避免"学不下去"）
- 数学要求高：金融工程、计算机、统计、物理、经济学——数学差硬冲会很痛苦。
- 背诵量大：医学、法学、生物——记忆力和耐心是门槛。
- 编程密集：计算机、软工、AI、数据科学——讨厌写代码的慎入。
- 需要动手/绘图：建筑、机械、设计、临床——空间感、手眼协调差要掂量。
- 怕血/怕尸体不能选临床、解剖相关；晕3D不适合某些设计。诚实评估孩子，别硬凑热门。

### 本硕博必要性（算总成本）
- 本科即可就业：计算机、软工、会计、护理、师范、多数工科。
- 强烈建议/必须读研：法学（法考+多数好岗要研）、金融（好岗要研+出身）、基础理学（数理化生）、心理学、生物。
- 必须读博/超长线：医学（临床5+3起）、科研路线、高校教职。
- 决策：如果家庭只能supporting到本科、或孩子不想长期读书，避开"不读研等于白读"的专业。

### 专业鄙视链与"天坑"真相
- "生化环材"（生物/化学/环境/材料）被称四大天坑：本科难就业、薪资低、劝退多——但**头部院校+读到博+进对行业（如材料进半导体、生物进创新药）也有出路**，是"普遍坑、局部金"。
- "天坑"是相对的：看院校层次、是否深造、进什么细分行业。skill 不简单贴标签劝退，而是讲清"这个专业在什么条件下是坑、什么条件下不是"。

### 新兴专业辨真伪（风口还是噱头）
- 近年高校跟风新设大量"人工智能""大数据""智能制造""区块链"专业。辨别：①师资是真有积累还是临时拼凑（查该校该方向的研究所/导师/学科基础）；②是不是依托学校原有强势学科升级来的（是=靠谱，凭空新设=存疑）；③首届招生没有就业数据参考，属"开荒一代"有风险。
- 真风口（国家重仓、缺口大）和蹭热点（学校招生噱头）要分清，结合就业手册的行业趋势判断。

### 性别视角（家长关心，中性呈现）
- 某些行业存在隐性性别偏好（如部分工科现场岗、地质野外作业），客观陈述就业环境，但不预设"女孩就不能学"——很多工科女生发展很好。把真实情况告诉家长，选择权交还。

## 八、更多家长视角维度（功利、现实、招生不说的）

### 身体硬门槛前置（选专业第一步就筛，不是事后提醒）
孩子的身体条件直接决定哪些专业根本碰不了，这一步在一切之前：
- 色盲/色弱→限报化学、生物、医学、材料、地质、计算机部分等（每年大量退档在此）。
- 视力→警校军校飞行技术有硬指标；身高→部分军警、空乘、师范个别地区。
- 其他→乙肝等传染病史部分专业限、单科成绩门槛、外语口试要求。
- 操作：选专业前先用孩子的体检结果对照《普通高等学校招生体检工作指导意见》逐条筛，把碰不了的专业先划掉，再谈别的。

### 专业的资格证书地图（未来吃饭的隐形门槛）
每个专业未来靠什么证吃饭，决定它的真实门槛：
- 法学→法考（不过基本白学）；医学→执业医师（5年起）；师范→教师资格证（相对易）；会计→CPA（极难但值钱）；建筑/土木→一建/注册建筑师；金融→CFA/FRM（加分项）；心理→咨询师资质。
- 报告要落地："这个专业未来吃饭要考〔X证〕，难度〔高/中/低〕，没证能不能干、能干什么。"——让家长知道专业的隐形门槛。

### 专业的性别就业真相（不预设、只陈述现实）
某些专业就业市场对性别有隐性偏好，如实告诉家长真实门槛，不预设"谁不能学"：
- 对女生有隐性门槛的：部分工科现场岗（野外/工地/值班）、地质采矿、某些重工。
- 对男生有隐性门槛的：护理（招男生少但近年缺口大反而吃香）、学前教育。
- 师范女生已饱和、男生稀缺反吃香（尤其中学理科、体育）。
- 陈述就业现实，选择权交还，很多"门槛"专业里逆势发展得好的也不少。

### 试错成本全景（进去发现不喜欢的所有后悔药）
把所有"补救路径"的可行性和成本汇总，让家长报考前就知道退路：
- 转专业（查该校政策，见p1/c5）→最常用，但热门难转。
- 辅修/双学位→补充第二技能，额外学分和精力成本。
- 考研换方向→最主流的"重新选择"，但跨考有难度、部分专业歧视跨考。
- 转学→现行制度基本不可行，别指望。
- 退学复读→核按钮，赌一年，见复读决策（按元铁律不替家长拍板）。
- 落地："如果孩子进去不喜欢，这个专业/学校的补救路径有〔哪些〕、各自代价〔多少〕"——报考前就把退路摊开。

### 大学隐性成本全清单（家境一般的硬约束）
学费之外，四年真实总开销，给家境一般的家庭算明白：
- 学费+住宿+生活费（按城市消费档次估）。
- 隐性大头：出国交换（几万到几十万）、考证培训班、考研班、电子设备（电脑/平板）、社交人情、谈恋爱、考公考编培训。
- 中外合作/高学费专业单列（见p1红线8）。
- 落地：给一个"四年大致总花费区间"，对家境一般的家庭这是真实决策约束。
